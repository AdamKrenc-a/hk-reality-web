# HK Reality Web

Webová aplikace pro HK Reality.

## Popis

Tento projekt obsahuje webové rozhraní pro HK Reality.

## Instalace

```bash
# Klonování repozitáře
git clone <repository-url>
cd web

# Instalace závislostí (pokud používáte npm/yarn)
npm install
# nebo
yarn install
```

## Spuštění

```bash
# Spuštění vývojového serveru
npm start
# nebo
yarn start
```
